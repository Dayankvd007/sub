## Decisions

| Decision | Date | Reason | Status | Related Phase |
| --- | --- | --- | --- | --- |
| Local backend | 7/22/2026 | Keeps credentials, persistence, and processing on the owner's computer and avoids cloud infrastructure for a one-user personal tool. | Current Decision | Phase 2 — Local Backend |
| FastAPI | 7/22/2026 | Provides a small typed Python HTTP layer around the validated translation engine and a simple local health check. | Current Decision | Phase 2 — Local Backend |
| SQLite | 7/22/2026 | Provides sufficient durable storage for one user, exact cache reuse, progressive results, and restart recovery without extra infrastructure. | Current Decision | Phase 2 — Local Backend |
| TypeScript extension | 7/22/2026 | Improves message-contract and lifecycle safety in a changing browser integration while remaining lightweight with Vite and plain HTML. | Current Decision | Phase 0 / Phase 3 |
| TextTrack synchronization | 7/22/2026 | Uses the media clock for cue activation while a custom RTL overlay controls Persian rendering; real playback behavior must still be validated. | Current Decision | Phase 3 — Chrome Extension |
| Polling instead of WebSockets | 7/22/2026 | Simple polling is adequate for one local client and incremental results, with less lifecycle and infrastructure complexity. | Current Decision | Phase 2 / Phase 3 |
| No React initially | 7/22/2026 | The MVP interface is too small to justify framework complexity; plain HTML and TypeScript are sufficient. | Current Decision | Phase 0 / Phase 3 |
| Translation provider/model: Anthropic `claude-sonnet-5` | 7/22/2026 | Strong Persian/multilingual quality and reliable strict-JSON output at moderate cost; provider abstraction keeps it a one-line change. Provisional pending owner's live comparison (P1-06). | Provisional Decision | Phase 1 — Translation Engine CLI |
| Prompt version v1 | 7/22/2026 | Versioned prompt loaded from a Markdown spec; participates in cache identity in Phase 2, so changes bump the version rather than editing in place. | Current Decision | Phase 1 — Translation Engine CLI |
